# ToDo App

A simple app for managing tasks with a visual representation of task completion.

## Features
- Add tasks easily through the app interface.
- Monitor the number of completed tasks using a heat map.

## Technology

1. Blok<br />
State management library that simplifies project state control and enhances readability.
2. Hive<br />
Lightweight and fast database solution for Flutter applications, providing local storage capabilities.
3. MVVM Architecture<br />
Model-View-ViewModel (MVVM) is a software design pattern that is structured to separate program logic and user interface controls.
